 Task 1: Hangman Game
   1.Goal: Create a simple text-based Hangman game where the player guesses a word one letter at a time.  
Features:
   1.Random Selection: Uses a predefined list of 5 words for gameplay.     
   2.Guess Limit: Players are strictly limited to 6 incorrect guesses.    
   3.User Interface: Basic console input/output for a clean, text-based experience.  
Key Concepts Used:
   1.random module for word selection.  
   2.while loops and if-else logic for game flow.  
   3.String and list manipulation.
